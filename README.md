
We define the Root Mean Square Error (RMSE) as: 

$$RMSE(X,h) = \sqrt{ {1 \over m}     \sum_{m_j} (h(x^(i^)) - y^(i^))^2}$$ 

where $$m$$ is the number of instances in the dateset under consideration. <br>
